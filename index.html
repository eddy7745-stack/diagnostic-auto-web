Questions fréquentes

Qu'est-ce qu'un code défaut OBD-II ?
Un code OBD-II est généré par l'ordinateur de bord lorsqu'il détecte une anomalie. Il commence par P (moteur), B (carrosserie), C (châssis) ou U (réseau) suivi de 4 chiffres.

Comment lire un code défaut sans valise ?
Vous pouvez utiliser un lecteur OBD-II Bluetooth avec une application Android. La prise OBD se trouve sous le tableau de bord côté conducteur, généralement près du volant.

Un code défaut = panne grave ?
Pas nécessairement. Certains codes sont mineurs (capteur légèrement hors tolérance), d'autres sont critiques (risque moteur). Cherchez votre code ici pour connaître la gravité exacte.
